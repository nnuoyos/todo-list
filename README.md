
# TO DO LIST

### 💻프로젝트 소개
투두 리스트 사이트

### ⏰작업 기간
- 5 days

### 🔗로직 & 기능

1. 유저가 값을 input에 입력한다
2. +버튼을 누르면 아이템이 추가된다. 할일이 추가 됨
3. 유저가 딜리트 버튼을 누르면 할일이 삭제 된다
4. 체크버튼을 누르면 할일이 끝나면서 밑줄이 생긴다
5. 진행 중 끝남 탭을 누르면 언더바가 이동한다
6. 끝난탭은 끝난 아이템만, 진행중탭은 진행중인 아이템만 보여준다
7. 전체탭을 누르면 전체 목록이 나온다
